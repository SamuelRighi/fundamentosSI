# Palavras ou expressões

Os termos a seguir são modelos de serviços em computação em nuvem, onde ao invés de você comprar servidor, programas ou infraestrutura, uma empresa pela internet ti dá isso tudo como serviço.

## PaaS - Plataforma como Serviço

- Você ganha uma plataforma pronta pra fazer e rodar aplicações sem se preocupar com nada, ideal pra dev focado em código
- Ex: Google Cloud Platform, Microsoft Azure

## IaaS - Infraestrutura como Serviço

- Vc aluga servidores, armazenamento e rede virtual e controla tudo (sistema operacional, aplicações, configurações do servidor) mas nn precisa comprar hardware físico
- Ex: Amazon Web Services (EC2), Google Cloud

## SaaS = Software como Serviço

- Programa pronto da internet
- Ex: Google Docs, Microsoft 365

## FaaS - Função como Serviço

- Roda funções específicas de código sem gerenciar servidor
- Ex: Amazon Web Services (Lambda), Google Cloud Platform (Cloud Functions)

## DaaS - Computador como Serviço

- Usa um computador virtual na nuvem como serviço
- Ex: Citrix, VMware

## DBaaS - Database como Serviço

- Banco de dados pronto, sem precisar mexer em servidor
- Ex: MongoDB (Atlas), Amazon Web Services (RDS), Google Cloud (Cloud SQL)

## SECaaS - Segurança como Serviço

- Bem auto-explicativo, né bobalhão (firewall, antivirus, essas tranqueiras ai)
- Ex: Cloudfare, Okta, Zscaler
